1 - Brush Stroke: Click "Draw Brush!" and use it on the canvas by clicking and moving the mouse.
2 - Erase Stroke: Click "Erase Brush!" and use it on the canvas by clicking and moving the mouse.
3 - Draw Line   : Click "Draw Line!" and use it on the canvas by clicking and moving the mouse. Release the mouse to finalize the line.
4 - Color Slider: Adjust the sliders to set the brush color, which will be displayed in the color box.
5 - Undo/Redo   : Click the Undo or Redo button to revert to or restore previous brush strokes.
6 - Select Layer: Choose the layer you want to brush on.
7 - Swap Layer  : Select the layers you want to swap.
8 - Save/Load   : To save the current grid with colors, click "Save" (it will save as save.json). To load a saved file, click "Choose File" and select the file.
